HTML Ordered List

An ordered list starts with the ol tag Each list item starts with the li tag. The list items are marked with numbers
        Attribute
        @compact it defines the list should be compacted 
        @reversed it defines that the order will be descending 
        @start it defines from which number or alphabet the order will start 
        @type it defines which type of the order you want in your list of numeric alphabetic or roman numbers
