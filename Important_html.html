HTML Block and Inline Element 

 Block Element A block element always starts on a new line and stretches out to the left and right as far as it can i.e it occupies 
    the whole horizontal space of its parent element
Block tag
Inline ElementAn inline element is the oppsite of the block level element it does not start on a new line and takes up only the necessary width 
    it only occupies the space bounded by the tag defining the HTML element, instead of breaking the flow of the content

HTML responsive web Design

            HTML responsive web Design refers to the changes in web pages to automatically resize,hide,show ,shrink and expands various HTML element
            These changes are depends on the devices and the screen size of the device.
        
            a)using HTML Viewportmate tag for responsive web Design:HTML viewport reprsents the area of display or screen visisble to the user.
            it varies from device to device this approach set the avaibale screen witdth as the 100% width of the web pages and adjust 
            accordingly to make it responsive.
        
            b)Using responsive images:
            Responsive images are just part of responsive website.Images that can changes their dimensions,scaling them up or down.
            
        

        c)CSS media QueriesThe media query in css is used to  create a responsive web design It means
        that the view of a web page differs from system to system based on screen or media types.The breakpoint specifies for what 
        device width size the content is just starting to break or deform
        media queries can be used to check many things 
        @width and height of the viewport
        @widht and height of the device
        @orientation
        @resolution

        
        d)Using Responsive layoutusing css display  property to make the page responsive.Display layouts like 
        flexbox,inline,blocks and grids  can be used to make design responsive.css flexbox proprerty auto adjust the content
        accordingly to the screen width.
        
using css grideThis approach uses a css display grid to create a 2D layout with other grid options
