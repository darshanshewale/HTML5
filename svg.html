HTML SVG:Scalable Vector Graphics. It basically vector based graphics i XML format.
    SVG graphics do not lose any quality if they are xoomed or resized. SVG can be animated.
    SVG images can be created and edited with any text editor
    SVG images can be searched, indexed,scripted and compressed
    SVG Images are scalable 
    SVG images can be printed with high quality at any resolution 

    Different between HTML SVG and HTML canvas  
    SVG is a language for describing in XML whereas canvas draws 2D graphics on the fly with javascipt 
    if SVG objects are changed browser automatically renders the shape whereas canvas is rendered pixel by pixel
    SVG is resolution independent whereas canvas is resolution-dependent 
    SVG supports event handlers whereas canvas doen't have support for event handlers.
