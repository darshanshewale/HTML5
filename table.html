HTML thead tag

The HTML thread tag is used to provide  a header to the group of content in an HTML table. combination the tbody and tfoot 
        element with the thread element can help to specify each part of a table i.e header,body and footer. by using these element, 
        browsers allow the table body to scroll separatly from the header and footer.
        the tread tag also supports the glabal attributes and event  attributes in HTML 
        the attribute values used with thread are deprecated in the HTML5.
